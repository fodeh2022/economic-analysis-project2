## 🧭 Project Overview

This project was developed as part of the **Business Economics** graduation requirements at **Birzeit University, Palestine (2020–2024)**.

The study focuses on analyzing economic and financial data to identify trends, evaluate market performance, and support data-driven decision-making. It applies quantitative and analytical methods — including statistical modelling in **SPSS** and data processing in **Excel** — to extract meaningful insights from real-world economic datasets.

Key areas of focus include wage determinants, foreign direct investment effects on the Palestinian economy, and stock market price behavior on the Palestine Exchange (PEX).

## 📈 Key Study: Impact of Higher Education on Wages in Palestine

| Detail | Value |
|--------|-------|
| Dataset | Palestinian Central Bureau of Statistics (PCBS), 2016 |
| Sample size | **23,342 individuals** |
| Method | Multiple Linear Regression (8 independent variables) |
| Model result | **R = 0.480** |
| Key finding | Education level and workplace location are the strongest wage predictors (p < 0.001) |
| Additional methods | ANOVA, descriptive statistics, cross-tabulation, frequency analysis |
